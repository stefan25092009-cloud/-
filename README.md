„Термохимия“ е интерактивен образователен сайт, който представя основните понятия за топлинните ефекти на химичните реакции. Чрез кратка теория, визуализации и тест с 30 въпроса потребителите могат да усвоят знания за екзотермични и ендотермични процеси, енталпия, калориметрия и приложенията на термохимията.
